# Entishar-Task
To test :

1-Open the project

2-Open the sql server object explorer window.

3-Copy the connection string of the local sql server in your machine.

4-In the appSettings.json , modify the cs with the new Connection string of your sql server.

5-In the Package manager console , type : add-migration Migration-Name to create the migration models . 

6- Then Type : Update-database to create the db.

7- Fill the db with dummy data .
